# `src/modeling/results_visualizer.py`

## Função

Visualização agregada de resultados de modelagem: ordenação de cenários (A–F, calculated, tsfusion), gráficos de barras/comparação, exportação de figuras para artigo/TCC.

## CLI

`python src/modeling/results_visualizer.py`

## Dependências

`matplotlib`, `pandas`, paths configurados no próprio módulo.
