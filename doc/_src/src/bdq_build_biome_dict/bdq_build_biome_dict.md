# `src/bdq_build_biome_dict.py`

## Função

Script de apoio para construir ou atualizar dicionário/auxiliar de bioma (apoio a filtros e consolidação BDQueimadas). Consulte o arquivo-fonte para argumentos exatos da CLI e colunas esperadas.

## Uso típico

Execução pontual quando a base de mapeamento bioma/município precisa ser regenerada a partir de dados brutos ou tabelas externas.
